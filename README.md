## Test Task
Develop an application based on the layout - an analogue of a timer in iOS or Android using React.
1. Ability to create an infinite number of timers;
2. Add timers;
3. Delete timers;
4. Start timers individually;
5. Start all timers;
6. Pause all timers;
7. Reset all timers.
